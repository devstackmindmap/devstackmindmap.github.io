AWS Lambda with Visual Studio and C#

Posted on 2021, Apr 09 2 mins read

AWS Toolkit for Visual Studio에는 Visual Studio용 AWS Lambda .NET Core 프로젝트 템플릿이 포함되어 있습니다. .NET Core CLI(명령줄 인터페이스)를 사용하여 Visual Studio에 내장된 Lambda 함수를 배포할 수 있습니다.

Visual Studio의 확장 관리로 갑시다.

Lambda로 검색 해서 “AWS Toolkit for Visual Studio 2017 and 2019"를 다운로드 받습니다.

다운로드 중이네요.

Modify Go Go.

열려 있는 Visual Studio 창들을 모두 닫습니다.

오오. 완료 되었습니다.

Visual Studio를 열면 AWS Getting Started 창이 열리고 Access Key, Secret Key를 넣고 Save and Close를 하니 !!

AWS Explorer 창이 뜨네요. 뭔가 강력 해보입니다.

새 프로젝트 만들기에서 “AWS Lambda Project with Tests (.NET Core-C#)” 으로 만들어 봅시다.

강력한 템플릿들이 있네요. 일단 Empty Function으로 만들어 봅시다.

자동으로 생성된 구성입니다.

함수를 작성하고 솔루션 탐색기 프로젝트에서 오른 클릭하여 “Publish to AWS Lambda…“를 통해 배포할 수 있습니다.

배포하기 전 해당 람다 함수의 Security Group과 Role을 추가 하였습니다.
람다 생성 시 준 권한들

  • AmazonRDSDataFullAccess
  • AWSLambdaBasicExecutionRole
  • AWSLambdaVPCAccessExecutionRole

Lambda 함수 업로드(Upload Lambda Function) 창에서 함수의 이름을 입력하거나 이전에 게시된 함수를 선택하여 다시 게시합니다. 그런 다음 [Next]를 선택합니다.

미리 생성한 Role과 Security Group을 설정하고 subnet도 설정하고 Upload 합니다.

Lambda 함수 구현과 API Gateway 설정을 진행합니다.

AWS-API-Gateway

dotnet lambda deploy-function
