Unity 해상도

Posted on 2021, Jun 06 2 mins read

Unity 해상도에 대해 아래 영상을 통해 배운 것을 정리 해보자

[고라니TV-게임개발 채널] 이거 하나면 모바일 해상도 걱정 끝! 16:9 고정해상도 만들기


현재 가장 많이 쓰는 해상도는 9:16!

나는 눕혀서 할꺼니까 1980:1080으로 !

기본 제공 되는 것으로 만드니 뭔가 해상도 숫자가 이상해서 Custom으로 만드니 오케이.

Constant Pixel Size 대신 Scale With Screen Size로 해주면 이미지 크기들이 일정하게 유지 된다고 한다.
그리고 Reference Resolution로 수정.

Match Width Or Height 가 Width에 가까우면 Shrink,
Height에 가까우면 Expand 랑 똑 같은 것
Expand로 하면 가장자리가 짤리지 않음
일단 Expand로 설정

이미지를 키워서 Canvas 크기에 맞춰보자.

메인카메라에 Viewport Rect 가 있다. 이것의 X, Y, W, H를 조절해보면 카메가의 크기를 조정할 수 있다.
CameraResolution 스크립트를 하나 만들어서 Main Camera에 넣어주자. 카메라가 여러개라면 다 넣어주자.

스크립트에 아래 코드 작성.

using UnityEngine;

public class CameraResolution : MonoBehaviour
{
    private void Awake()
    {
        Camera camera = GetComponent<Camera>();
        Rect rect = camera.rect;
        float scaleheight = ((float)Screen.width / Screen.height) / ((float)16 / 9);
        float scalewidth = 1f / scaleheight;
        if (scaleheight < 1)
        {
            rect.height = scaleheight;
            rect.y = (1f - scaleheight) / 2f;
        }
        else
        {
            rect.width = scalewidth;
            rect.x = (1f - scalewidth) / 2f;
        }
        camera.rect = rect;
    }
}

Screen 에서 나온 것은 기기의 해상도. 정확한 값을 위해 float 사용. 그리고 뒤에 16/9 넣어준 값은 우리가 고정한 비율

float scaleheight = ((float)Screen.width / Screen.height) / ((float)16 / 9);

scaleheight 값이 1보다 작거나 큰 값이 나온다.
1보다 작은 경우는 날씬한 경우.
이렇게 설정하고 난뒤 Rect 툴로 두고 조정하고 앵커를 조정하면 안된다.

이미지를 저런 식으로 올리고 빌드 한번 해보자.

File > Build Settings > Player Settings > Player > Resolution and Presentation 에 가면

Portrait : 핸드폰 세로로 들었을 때

Portrait Upside Down : 핸드폰 세로로 거꾸로 뒤집었을 때

Landscape Right : 홈 버튼이 오른쪽으로 갔을 때

Landscape Left : 홈 버튼이 왼쪽으로 갔을 때

나는 가로 게임을 하려고 하니

Auto Rotation 에 Landscape Right와 Landscapde Left 만 체크 !!

그리고 Company Name 과 Other Settings에 Package Name 수정.
