GameLift Fleet 인스턴스 원격 접속

Posted on 2021, Apr 01 One min read

연결할 인스턴스 ID 찾기

aws gamelift describe-instances --fleet-id FLEET-ID

인스턴스에 대한 액세스 자격 증명을 요청

aws gamelift get-instance-access --fleet-id FLEET-ID --instance-id INSTANCE-ID

원격 연결을 위한 포트 열기

aws gamelift update-fleet-port-settings --fleet-id FLEET-ID --inbound-permission-authorizations “FromPort=3389,ToPort=3389,IpRange=LOCAL-PUBLIC-IP/32,Protocol=TCP”
