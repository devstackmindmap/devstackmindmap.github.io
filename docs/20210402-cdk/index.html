CDK in C#

Posted on 2021, Apr 02 One min read

환경

Windows / Visual Studio 2019 / C#

“백문이 불여일견이요, 백견이 불여일각이며, 백각이 불여일행.”

  1. mkdir HelloServerlessCDK
  2. cd HelloServerlessCDK
  3. cdk init app --language csharp (디렉토리가 비어 있어야 한다)
  4. [project]\src 에서 *.sln 열기
  5. Ctrl + F5, 성공하면 출력이 없어야 한다.
  6. Ctrl + `, 터미널을 연다, 개발자 명령 프롬프트 (개발자 PowerShell 말고)
  7. cd ..
  8. NuGet에서 기본 설치되어 있는 Amazon.CDK를 업그레이드 하거나 삭제 후 최신버전 설치
  9. NuGet에서 Amazone.CDK.AWS.S3 설치
  10. Infra 코딩 (테스트 코드에 BucketName 수정) 자동으로 생성된 Stack 클래스의 생성자 함수 내부에 인프라 코드 입력
var bucket = new Bucket(this, "MyBucket", new BucketProps
{
    Versioned = false,
    BucketName = "test-bucket"
});
  1. cdk bootstrap (Project root)
  2. cdk synth or Ctrl + F5
  3. cdk deploy
  4. cdk destroy
