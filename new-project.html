    프로젝트 제안👆🏽

    기존 서비스 했던 게임(나이트런:레콘키스타, 현재는 서비스 종료)에서 인게임 부분을 가져와서 해당 인게임에 어울리는 새로운 세계관, 스토리, 캐릭터, 컨텐츠, BM을 입히는 신규 프로젝트 제작을 제안합니다
    신규 프로젝트에 대한 구체적인 기획서를 공유하는 것이 아니라 구현된 인게임의 가능성을 보시고 함께 구상, 구현해 보면 좋을 것 같습니다.

    기존 게임의 외관은 서브컬처, 모바일 수집형 게임으로 보이나 게임성은 하스스톤, 클래시 로얄을 섞은 전략 PVP 라고 보실 수 있습니다.
    신규 프로젝트는 여기에 PVE를 강화하면 좋을 것 같습니다.

    만약 테스트해보시게 된다면 2명 이상의 플레이어가 자신만의 덱을 구성하여 PVP 테스트, 여러 판 해보시길 바랍니다.

    APK 링크(튜토리얼 있는 버전)
    APK 링크(튜토리얼 없는 버전)

    테스트해보실 수 있게 APK를 준비했습니다. 개인 PC에서 데이터를 다운로드하기 때문에 설치 후 실행하면 진행되는 데이터 다운로드 속도가 느린 점 양해 부탁드립니다.
    서버 또한 개인 PC에 세팅되어 있어 본의 아니게 서버가 꺼져있을 수 있으니 연락 주시면 서버를 오픈하도록 하겠습니다.

    만약 모바일 기기에서 전투 진입이 안된다면 아래 LD플레이어로 테스트 부탁드립니다. LD플레이어 다운로드

    덱 편집 설명👆🏽


    1. 나이트리그로 진입합니다.


    2. 덱 편집으로 진입합니다.


    3. 드래그 앤 드롭으로 유닛 선택이 가능합니다. 3개의 유닛을 선택합니다.


    4. 유닛의 스탯과 패시브, 스킬 정보를 확인할 수 있습니다.


    5. 스킬 편집으로 진입합니다.


    6. 유닛을 선택하여 해당 유닛의 스킬로 이동하고 스킬을 선택하면 스킬의 상세정보를 확인할 수 있습니다.




    7. 3개 유닛의 보유한 스킬 중 총 8개를 선택할 수 있습니다.
    궁극 스킬은 최대 1개, 특수 스킬은 스킬별 1개, 일반 스킬은 제한없이 선택 가능합니다.


    8. 드래그 앤 드롭으로 8개의 스킬 선택이 끝나면 스킬 편집 완료, 덱편집 완료로 구성을 종료합니다.
    필요하다면 여러 슬롯에 다양한 덱을 구성해 놓을 수 있습니다.




    9. 덱편집이 완료되면 스타트 버튼을 눌러 게임을 시작할 수 있습니다.
    기본적으로 유저 간 매칭이 되며 현재 10초간 유저 매칭이 안되면 AI와 매칭이 되도록 세팅하였습니다.

    1. 게임에 진입하면 아래와 같습니다.


    2. 승리조건
    인게임 UI 설명 전 게임의 승리 조건에 관해해 설명드리겠습니다.

  • 기본 게임 시간 3분 안에 상대 유닛 3명을 모두 쓰러뜨리면 승리합니다.
  • 만약 3분이 지났을 때 나와 상대의 유닛 수가 같다면 연장전이 주어지고 연장전 시간 내에 1명을 먼저 쓰러뜨리면 승리합니다.
  • 만약 3분이 지났을 때 나와 상대의 유닛 수가 같다면 연장전이 주어지고 연장전 시간 내에 1명을 먼저 쓰러뜨리면 승리합니다.

    • 3. 인 게임 UI 상단부에 대한 설명입니다.

  • 각 유닛 머리위에는 3가지 숫자가 있습니다.
  • 게이지바 위에 있는 숫자는 HP입니다.
  • 게이지바 아래에 있는 숫자는 기본 공격력입니다.
  • 게이지바 왼쪽에 있는 숫자는 일반 공격 쿨타임으로 쿨타임이 0이 되면 일반 공격을 합니다. (정확하게는 일반 공격이 대기열에 들어갑니다.)
    •


    3. 인 게임 UI 하단부에 대한 설명입니다.

  • 덱 편집된 스킬 8개가 랜덤하게 섞여 그 중 4장이 사용할 수 상태(핸드)로 시작합니다.
  • 스킬을 사용하면 사용 스킬은 카드 대기열 맨 뒤로 가며 카드 대기열 맨 앞에 있는 스킬이 사용할 수 있게 됩니다.
  • 카드 대기열 맨 앞의 스킬은 UI 오른쪽 "NEXT SKILL" 형태로 보이게 됩니다.
  • 유닛이 죽게 되면 해당 유닛의 스킬 또한 사용 불가 상태로 내 핸드 및 카드 대기열에서 모두 사라집니다.
  • 가운데 아래 게이지는 "노심"이라 불리며 스킬을 사용할 수 있는 Point 입니다.
  • 각 스킬 오른쪽에 있는 숫자가 필요 "노심" 수입니다.
    •



    4. 기타 인게임 설명

  • 인게임의 중요한 시스템 룰 중 하나인 대기열에 대해 설명하겠습니다.
  • 대기열에는 크게 일반 공격과 스킬이 들어가게 됩니다.
  • 일반 공격은 인게임 UI 상단부, 각 유닛 위의 게이지 왼쪽 숫자가 0이 되면 대기열로 들어가게 됩니다.
  • 스킬은 유저가 사용하면 대기열로 들어가게 됩니다.
  • 특수한 규칙 중 하나는 대기열에 스킬이 하나라도 있다면 일반 공격보다 항상 우선합니다.
  • 스킬 우선 규칙 외에는 모두 사용한 순서대로 발동되는 것이 규칙입니다.
  • 궁극 스킬과 특수 스킬의 모션 중에는 전체 게임 시간이 멈추게 됩니다.(게임 시간, 모든 쿨 타임 등 시간 관련된 것 모두)
  • 일반 스킬에 모션이 있어도 모든 게임 시간은 진행됩니다.

    기존 게임 전략 요소👆🏽

  • 유닛의 패시브, 스탯
  • 평타형 덱, 스킬 덱, 성장형 덱, 반격 덱, 특수 덱, 쉴드 덱, 스탯 덱 등
  • 궁극 스킬이 어떤 유닛의 어떤 궁극기인지에 따라 양상이 달라짐
  • 상대의 궁극 스킬을 예상하고 대응
  • 8개의 스킬 구성, 스킬 회전
  • 핵심 유닛 몰빵이냐, 분산이냐

    • 포스트모템👆🏽

    저희는 나이트런이라는 색깔이 강하고 팬층이 확실한 IP를 가지고 있었고 가능한 선에서 마케팅도 해당 팬층을 향하고 있었습니다.
    IP를 제외하고 인게임만 본다면 마케팅 대상도 달라야 했고 전체적인 게임 제작 방향성도 달랐어야 합니다.
    IP를 가지고 있는 상태에서 가장 큰 실패 원인은 내, 외부에서 모두 바라는 RPG로 게임을 만들었어야 했는데 그렇게 하지 못한 점입니다.
    운영진은 투자받고 어느 정도 개발이 진행된 상황에서, 전환은 어렵다고 생각했지만, 당시로 돌아간다면 과감하게 뒤집어서 새로 제작에 들어가야 했습니다.
    웬만한 국내외 퍼블리셔를 모두 만났지만 그분들도 저희 팀이 기존에 했던 갓 오브 하이스쿨과 같은 RPG를 생각 했고 만들고 있던 나이트런:레콘키스타 게임의 외관도 RPG 였습니다.

    첫째, 그때는 모바일 수집형 RPG가 맞았다.

    기존 게임의 아트 퀄리티가 너무 높았습니다.
    규모에 맞지 않는 수준이었고 생산성이 너무 떨어졌습니다.
    Spine, Live2D와 같은 툴을 쓰지 않고 고 프레임의 아트를 전부 그렸습니다.
    라이브 직전과 라이브 모두에서 이 부분은 큰 벽이었습니다.

    둘째, 아트 퀄리티가 높았고 규모에 맞지 않아 생산성이 떨어졌다.

    같은 맥락에서 유저들이 원하던 PVE 컨텐츠를 생산하기도 쉽지 않았습니다.
    라이브 중 급하게 스토리 컨텐츠를 대신할 PVE 컨텐츠를 추가했지만 역부족이 었습니다.

    셋째, 스토리(PVE) 컨텐츠의 부재.

    그렇다면, 인게임에 어울리는 신규 프로젝트를 만든다면 고민해야 할 핵심 부분은 무엇일까요.
    계속되는 신규 유닛과 신규 스킬 업데이트로 게임의 재미와 수명을 유지, 발전할 수 있을지가 핵심 이슈가 될 것 같습니다.
    인게임 rule이 안정적이라면 다양한 시도가 가능하겠다고 생각합니다.
    이제 9년이 된 하스스톤을 참조할 수도 있고 완전히 새로운 관점에서 생각해 볼 수도 있을 것 같습니다.

    결론👆🏽

    IP를 제외하고, 다양한 PVP 모드와 글로벌 인프라 구성으로, 다양한 PVE 인게임 장치를 더 개발하여 신규 게임을 만들면 괜찮은 전략 게임을 만들수 있다고 생각합니다. 인게임 테스트 후 가능성이 보인다면 새로운 세계관, 스토리, 캐릭터, 콘텐츠, BM으로
    글로벌, 멀티 플랫폼, 풍부한 PVE와 PVP를 갖춘 신규 전략 게임으로 완성시켜 보면 어떨까 합니다.
    감사합니다.
