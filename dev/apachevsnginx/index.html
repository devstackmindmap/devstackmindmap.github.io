Apache vs Nginx

PHP와 Apache로 웹 서버를 만들고 있었다. 그때 대표님이 nginx를 도입해보라고 권유하셔서 nginx 검토 및 apache에서 nginx로 교체하였다.

다시 한번 어떠한 차이가 있는지 확인해 보자.

Apache

  • 웹 서버 데몬
  • HTTP 웹 서버
  • 동적 로딩 모듈 기반 시스템

Nginx

  • 비동기식 이벤트 처리 중심의 아키텍처
  • 경량의 자원활용
  • 물리적인 하드웨어로 쉽게 확장
  • 정적인 컨텐츠들을 빠르게 처리
  • 동적 처리를 별도로 담당하는 소프트웨어 스택들과 연계해서 고성능 서비스 제공
  • 웹 서비스 뿐 아니라 프록시 서비스 기능에도 많이 사용
